## Preprocessing
1. remove extra columns
2. clean comments
  - remove emojies
  - remove URLs
  - remove Hashtag
  - remove empty spaces
  - remove HTMLs
  - Normalization (Hazm)
 3. classify rates
  - rate >= 3 ==> 1
  - rate < 3 ==> 0
